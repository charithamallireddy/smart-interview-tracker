 Smart Interview Tracker

A simple Python project to track interview preparation progress.

 Features
- Save number of problems solved
- Track confidence score
- Get topic recommendations
- View saved history
- Interactive UI using Streamlit

 Tech Stack
- Python
- MySQL
- Streamlit
- Git & GitHub

 Files
- `app.py` → Main Python app
- `db.py` → Database connection
- `analytics.py` → Analytics logic
- `recommendation.py` → Recommendation system
- `streamlit_app.py` → Streamlit UI

 How to Run
```bash
pip install streamlit mysql-connector-python
streamlit run streamlit_app.py
```
